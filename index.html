Conceito de banco de dados

Conceito

SGBD (Sistema de Gerenciamento de Banco de Dados) é uma estrutura de software que permite que dados sejam armazenados, organizados, protegidos, atualizados, acrescentados, excluídos e acessados sempre que necessário, devendo corresponder à demanda que a aplicação que o utiliza exige.

BD relacionais

Esse tipo é muito utilizado em sistemas de ERP (gestão de empresas), CRM (relacionamento com o cliente), controle financeiro e tantas outras aplicações porque os dados são estruturados em tabelas cujas colunas e linhas se relacionam. Uma loja, por exemplo, pode ter uma tabela de clientes, outra para controle de estoque, uma terceira para fluxo de caixa e assim por diante.

Tipos de dados e chave-primária

Tipos de dados

DECIMAL(M,D) – Ponto decimal com M dígitos no total (precisão) e D casas decimais (escala); o padrão é 10,0; M vai até 65 e D até 30.;
FLOAT(M,D) – Ponto flutuante com precisão M e escala D; o padrão é 10,2; D vai até 24.
CHAR(M) – String que ocupa tamanho fixo entre 0 e 255 caracteres BOOL / BOOLEAN – Valores binários 0 / 1; Na verdade, é um alias para o tipo TINYINT(1).
VARCHAR(M) – String de tamanho variável, até 65535 caracteres.
BLOB / MEDIUMBLOB/ TINYBLOB – Campo com tamanho máximo de 65535 caracteres binários; ‘Binary Large Objects’, são usados para armazenar grandes quantidades de dados, como imagens.

Chave-primária

Chaves primárias (em inglês, Primary keys ou "PK"), sob o ponto de vista de um banco de dados relacional, referem-se aos conjuntos de um ou mais campos, cujos valores, considerando a combinação de valores em caso de mais de uma chave primária, nunca se repetem na mesma tabela e, desta forma, podem ser usadas como um índice de referência para criar relacionamentos com as demais tabela do banco de dados (daí vem o nome banco de dados relacional). Portanto, uma chave primária nunca pode ter valor nulo, nem repetição.

Gerenciador de BD e Linguagem SQL

Gerenciador de BD

Gerenciadores de bancos de dados MySQL, são plataformas nas quais voce pode conectar remotamente seu banco de dados, sendo possivel assim alterar, inserir, deletar, e listar os dados que nele estão guardados.

Linguagem SQL

A linguagem SQL, é a linguagem utilizada para poder fazer as alteraçoes basicas no Banco de dados, e dentro dessa linguagem temos os comando basicos mais importantes e mais utilizados:
Select - Comando utilizado juntamente com seus parametros para retornar uma informaçao especifica;
Insert - Comando utilizado para inserir algum novo dado em uma tabela do Banco;
Update - Comando utilizado para atualizar algum dado especifico desejado;
Delete - Comando utilizado para deletar um ou mais dados do Banco de dados;
